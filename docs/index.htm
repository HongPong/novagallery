Features

  • Albums: Directories are rendered as albums
  • Preview images for albums: Automatically generate an album preview image
  • Photo overview: All photos are displayed sorted by creation date
  • Lightbox: View your photos in large format, hiding everything that is distracting
  • Photo slideshow: All photos in an album, can be displayed and presented via click (or keyboard) in the lightbox view
  • Photo Upload: Simply upload the photos into the gallery folder (e.g. via FTP) and the photos are already online
  • Thumbnails: Automatic generation of thumbnails
  • File compression: Photos with high file size, are automatically compressed with very little loss.
  • Custom Themes: With just a little knowledge of CSS and HTML, the look can be customized.
  • JPEG image format: Currently JPG images are supported (.jpg, .jpeg, .JPG, .JPEG).

Docs

Requirements

  • Web space with PHP5.6 or PHP7.x
  • Apache, nginx or other web server
  • Currently supported file format for photos: jpg/jpeg

Installation

  • Upload files on Webspace
  • Change name and description in config/site.php
  • Create folders in galleries (used as albums)
  • Upload your images to the albums
  • Enjoy your new beautiful photo gallery

Manage Images

  • Add Photos: Just upload the new photo to the server into the correct album (e.g via FTP)
  • Delete a Photos: Just delete the photo at the server
  • Delete a Album: Just delete the whole album (directory) at the server

Note

  • The first time you open an album, the thumbnails are generated. This may take a some time. With every further request (across visitors) the generated thumbnails are used and the page speed is faster.
